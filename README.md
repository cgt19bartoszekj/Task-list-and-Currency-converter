## Instalation

1. Clone the repository:

   ```
   git clone https://github.com/jakub-bartoszek/task-list.git
   cd task-list
   ```

2. Install dependencies:
   ```
   npm install
   ```
3. Run application:
   ```
   npm start
   ```
